This project demonstrates a complete machine learning workflow on the classic Iris dataset, including data exploration, preprocessing, model training, and evaluation using multiple classification algorithms.


Project Overview
The goal of this project is to classify Iris flowers into one of three species:
(1) Iris-setosa
(2)Iris-versicolor
(3)Iris-virginica

Based on four input features:
Sepal Length
Sepal Width
Petal Length
Petal Width

Technologies & Libraries Used
Python
NumPy
Pandas
Matplotlib
Seaborn
Scikit-learn

Dataset
Dataset used: Iris.csv
Number of samples: 150
Features: 4 numerical features
Target: Species (categorical)

Exploratory Data Analysis (EDA)
The following visualizations were used:
Pair plots to understand feature relationships
Correlation heatmap to observe feature correlations
Dataset statistics and structure using info() and describe()

Machine Learning Models Implemented
Logistic Regression
K-Nearest Neighbors (KNN) – k = 5
Decision Tree Classifier

Model Evaluation
Evaluation metrics used:
Accuracy Score
Confusion Matrix
Classification Report (Precision, Recall, F1-score)
